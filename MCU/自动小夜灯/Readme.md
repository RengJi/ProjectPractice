## 功能
**功能描述**

- 可以用红外遥控开关小夜灯，关闭小夜灯则小夜灯不在开启LED，出厂默认开启小夜灯
- 小夜灯具有两种模式，一种**感应模式**和一种**常亮模式**， **常亮模式**下小夜灯不在感应外部信息，保持常亮，出厂模式下是**感应模式**
- 小夜灯具有记录功能，小夜灯的**[开/关]**和**模式**会被记录下来，掉电不丢失

**感应模式**
1. 光照弱才启用小夜灯，光照强则不启用
2. 启用小夜灯之后，噪声到达一定程序，灯开启10秒
3. 启用小夜灯之后，有人接近，灯开启10秒

## 方案设计
|模块|外设|占用IO口|
|-|:-|-|
|光敏传感器|ADC1通道1|GPIOA1|
|声音传感器|EXTI外部中断|GPIOB1|
|人体传感器|EXTI外部中断|GPIOB4|
|12VLED灯|12V继电器|GPIOB8|
|红外接收器|TIM2定时器中断|GPIOA8|
|蜂鸣器|GPIO|GPIOB12|
|EEPROM|I2C1|GPIOB6,GPIOB7|
