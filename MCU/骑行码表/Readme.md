## 功能
**功能描述**
- 支持两个按键切换
	1. 按键1，开始记录和停止记录功能
	2. 按键2，支持切换OLED屏幕显示内容
- 实时在OLED上面显示三页功能
	1.码表上电之后默认是欢迎界面，此时码表状态为**空闲**，按下按键1之后，表示开始骑行，此时码表状态为**骑行**，实时监测数据，然后显示在OLED屏幕上
	2.OLED信息显示页1：实时时间、实时速度、骑行时间和骑行距离
	3.OLED信息显示页2：最高时速、平均时速、消耗的卡路里和温湿度
	4.按下**按键2**可以切换显示信息
	5.**骑行**状态中，按下**按键1**，屏幕数据停止，不再计算，此时码表状态为**停止**
	6.**停止**状态中，再次按下**按键1**，状态切换回**空闲**，屏幕显示欢迎页面
- 可以估算消耗卡路里和测量温湿度
- 骑行过程中可以保存经度和纬度到flash中
- 骑行结束的同时，将总结信息开始时间，结束时间，总耗时，总距离，最高时速，平均时速，消耗的卡路里，平均湿度，平均温度保存到flash中
- PC端向码表发送get_data，可以获取保存的日志
## 方案设计
|模块|外设|占用IO口|
|-|-|-|
|按键1，按键2|GPIO|GPIOC14，GPIOB0|
|计时功能|TIM2|无|
|GPS接收|USART3|GPIOB11|
|OLED屏幕|I2C1|GPIOB6，GPIOB7|
|温湿度传感器|I2C1|GPIOB6，GPIOB7|
|传输日志|USART1|GPIOB9，GPIOB10|
|flash|SPI1|GPIOA4，GPIOA5，GPIOA6，GPIOA7|
