# ProjectPractice

Some projects I practiced by myself.

## Projects

| Project | Description | Language |
|---------|-------------|----------|
| [Calculator](./calculator/) | A simple command-line calculator supporting basic arithmetic operations | Python |
| [Todo List](./todo_list/) | A command-line todo list manager with add, complete, and delete features | Python |
| [Data Structures](./data_structures/) | Implementations of linked list, stack, and queue from scratch | Python |
| [Sorting Algorithms](./sorting_algorithms/) | Common sorting algorithms: bubble, selection, insertion, merge, and quick sort | Python |

## How to Run

Each project is self-contained in its own directory. Navigate into a project folder and run the main Python file:

```bash
cd calculator
python calculator.py

cd todo_list
python todo.py

cd data_structures
python data_structures.py

cd sorting_algorithms
python sorting.py
```

## Testing

Each project includes a test file. Run tests with:

```bash
python -m pytest <project_dir>/
```
